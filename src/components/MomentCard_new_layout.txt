# Layout mới cho MomentCard

## Thứ tự hiển thị:
1. Location Section (nếu có) - Ngoài card
2. Caption (nếu có) - Ngoài card  
3. Media Section với overlays:
   - Author Info Overlay (góc trên trái) - Trong ảnh
   - Interaction Overlay (giữa phía dưới) - Trong ảnh
   - Media Indicator (nếu nhiều ảnh) - Trong ảnh
4. Meta Section (time + category) - Ngoài card dưới ảnh

## Author Info Overlay style:
- position: absolute, top: 16, left: 16
- flexDirection: row
- backgroundColor: rgba(0, 0, 0, 0.5)
- borderRadius: 20
- padding: 8
- Avatar: 36x36px
- Text màu trắng
- Icon public/private màu trắng
- Menu button (3 chấm) màu trắng
